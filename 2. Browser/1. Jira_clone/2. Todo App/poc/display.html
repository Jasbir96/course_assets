Block Elements

The elements which occupy the complete horizontal space and does not let any other element occupy their horizontal space even if its empty is called an block level element. We can specify height and width properties for these elements. Example - div, p

Inline Elements

These elements let other elements occupy their horizontal space but we cannot specify height and width for such elements. Example - span
